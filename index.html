  1. O que é HTML?

    (voltar ao topo)

    • HTML significa "HiperText Markup Language", que traduzindo para o português é "Linguagem de Marcação de HiperTexto", ele serve para dar significado para a informação das páginas web, o HTML serve como o esqueleto de uma página web, incluindo. Vale lembrar que linguagem de marcação é diferente da linguagem de programação...
